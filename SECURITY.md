# Security and Privacy

This repository is public. Do not open issues or pull requests containing secrets, credentials, private source material, customer data, employer data, personal data, private workspace paths, screenshots of private systems, or confidential documents.

For content problems, use a GitHub issue or pull request with public-safe details only.

For security-sensitive concerns about this repository, contact the maintainers through GitHub without posting exploitable details publicly.

For security issues in OpenAI, Codex, MCP servers, browser tools, or third-party projects, use the responsible disclosure process for the affected project. This repository is an independent directory and does not operate those services.
