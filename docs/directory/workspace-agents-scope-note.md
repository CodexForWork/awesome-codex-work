## Work-First Interpretation

For office and team workflows, Workspace agents point to a useful pattern:

1. Start with a repeatable work process.
2. Identify the source systems and tools involved.
3. Define what the agent may read.
4. Define what it may write or send.
5. Add approval steps for high-impact actions.
6. Review outputs before they become business records.

That pattern is useful even when a team is not ready to build or deploy a shared agent.
