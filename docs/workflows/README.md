# Workflow Templates

These templates are copy-ready starting points for Codex-powered work. They are not finished outputs and not official OpenAI workflows.

Each template includes:

- audience;
- work problem;
- inputs;
- copy-ready Codex instruction;
- expected output;
- human review checklist;
- failure modes;
- source boundary.

## Templates

| Template | Use it for |
|---|---|
| [Daily Work Brief](daily-work-brief.md) | Turning scattered updates into a one-page working brief. |
| [Research to Decision Memo](research-to-decision-memo.md) | Turning source notes into a decision-support memo. |
| [Spreadsheet Cleanup Plan](spreadsheet-cleanup-plan.md) | Planning workbook cleanup before editing. |
| [Meeting Notes to Action List](meeting-notes-to-action-list.md) | Extracting commitments, owners, due dates, and open items. |
| [Review-Ready Deck Outline](review-ready-deck-outline.md) | Planning a slide structure before making a deck. |
| [Workflow Audit and Automation Spec](workflow-audit-and-automation-spec.md) | Mapping a manual process and deciding what can safely be automated. |

## Review Rule

Before using any output professionally, run [Human Review Before Use](../review-checklists/human-review-before-use.md).
