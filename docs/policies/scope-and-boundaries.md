# Scope and Boundaries

> **CodexWork is independent and unofficial. It is not affiliated with, endorsed by, sponsored by, or connected to OpenAI. Codex and related marks are trademarks of their respective owners.**

This repository is an independent public directory and playbook for Codex-powered work.

## In Scope

A resource is in scope when it has a meaningful relationship to Codex or Codex-style work and helps produce reviewable work.

Examples:

- official OpenAI Codex documentation;
- OpenAI Academy Codex for Work materials;
- Codex app, CLI, IDE, web/cloud, and GitHub integration resources;
- `AGENTS.md`, skills, plugins, automations, MCP, browser, and computer-use patterns;
- official or maintained repositories relevant to Codex-powered work;
- workspace-agent patterns when sourced and clearly scoped;
- workflow templates and review checklists;
- source-traceability methods;
- privacy, approval, and human-review patterns;
- practical office-work examples.

## Out of Scope

- Generic AI tool links without a Codex relationship.
- Unsourced claims about performance, pricing, availability, or access.
- Provider-fandom or anti-provider framing.
- "Free Codex", "bypass OpenAI", "replace OpenAI", or similar claims.
- OpenAI logos, official-style visuals, or wording implying affiliation.
- Private workspace details, customer data, employer names, credentials, logs, screenshots, or internal files.
- Auto-posting, outreach, or publication workflows outside this GitHub repo.

## Compatible Tools and Providers

Compatible tools may appear when they help explain or improve Codex-powered work. They should be task-led and clearly bounded by quality, privacy, review, and terms-of-use considerations.

OpenAI remains the default official route and quality benchmark for Codex.

## Public Language

Prefer:

- "Codex-powered work";
- "independent guide";
- "workflow directory";
- "reviewable work";
- "source-backed playbook";
- "human-reviewed work products";
- "workspace-native agent apps" when used softly as an emerging pattern.

Avoid:

- "official";
- "certified";
- "endorsed";
- "approved by OpenAI";
- "OpenAI partner";
- "first ever";
- "invented by us";
- "bypass OpenAI";
- "cheap replacement";
- "free Codex".

## Approval Boundary

This repo may publish GitHub content in this repository. It does not authorize posting, sending, deploying, or publishing to Reddit, X/Twitter, LinkedIn, Substack, Hacker News, CodexWork.com, or any other external surface.
